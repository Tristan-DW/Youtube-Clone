##  Overview

**Youtube-Clone** is a html/css youtube clone. aesthetics only. Built to be fast, reliable, and easy to use.

---

##  Features

- **Easy Setup** - Get up and running in minutes
- **Modern Design** - Clean and intuitive interface
- **Well Documented** - Clear instructions and examples
- **Open Source** - Free to use and contribute

---

##  Quick Start

```bash
# Clone the repository
git clone https://github.com/Tristan-DW/Youtube-Clone.git
cd Youtube-Clone

# Open in browser
open index.html
```
